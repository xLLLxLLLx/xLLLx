## Github教程

$Github$是个好东西QwQ

存代码不用U盘~~爸爸妈妈再也不用担心我的U盘弄丢没有备份啦!~~

![](https://img2018.cnblogs.com/blog/1624986/201904/1624986-20190416201929845-284417652.png)

- 创建github账号
- 创建仓库
- 输入命令

> git clone https://github.com/your_user_name/your_git_name

- 克隆完成之后

> cd your_git

- 添加路径

> git add*

> git status

输入用户名 密码就完成了！

----

- 上传文件

> git add *

> git commit -m “add new files”

输入用户名 密码就完成了！
